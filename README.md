# Telegram Mini App для записи на услуги

Мини-приложение для салона красоты, мастера или любой услуги с записью по календарю.

## Что уже есть

- Клиентская страница Mini App: выбор услуги, календарь, свободные и занятые слоты.
- Админка: добавление услуг, длительность услуги, цена, список записей.
- Проверка пересечений: если услуга занимает 90 минут, заняты будут все пересекающиеся слоты.
- Простое JSON-хранилище в `data/db.json`.
- Без внешних npm-зависимостей.

## Запуск

```bash
npm start
```

Открой:

- Mini App: `http://localhost:3000`
- Админка: `http://localhost:3000/admin.html`

PIN администратора по умолчанию: `1234`.

Можно задать свой:

```bash
$env:ADMIN_PIN="9876"
npm start
```

## Как подключить к Telegram

1. Создай бота в `@BotFather`.
2. Выложи проект на HTTPS-хостинг: Render, Railway, Fly.io, VPS, Vercel serverless adapter или другой Node.js хостинг.
3. В `@BotFather` открой `/mybots`.
4. Выбери бота.
5. `Bot Settings` -> `Configure Mini App` -> `Enable Mini App`.
6. Укажи HTTPS-ссылку на главную страницу приложения.

## Для реального запуска

Перед продакшеном лучше заменить JSON-файл на базу данных: PostgreSQL, SQLite или Supabase.
Также стоит добавить авторизацию администратора через Telegram ID или полноценный пароль.
